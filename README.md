## Начало работы

## Сборка и запуск - проект работает на последней версии node.js 22
- `npm ci`: устанавить зависимости;
- `npm start`: запуск версии для разработки;
- `npm run build`: сборка продакшн-версии;
- `npm run lint`: линтинг.
